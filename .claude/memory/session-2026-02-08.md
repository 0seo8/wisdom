# 지혜의밭(Artswisdom) 웹사이트 리빌드 프로젝트 세션

## 날짜: 2026-02-08

## 프로젝트 개요
- **프로젝트명**: wisdom (지혜의밭 웹사이트 리빌드)
- **기술 스택**: Next.js 16, TypeScript, Tailwind CSS 4, Framer Motion
- **원본 사이트**: https://artswisdom.com/

## 수행한 작업

### 1. MoAI-ADK 설치 및 설정
- moai-adk v2.0.1 설치 (`~/.local/bin/moai`)
- 프로젝트 초기화 (`moai init wisdom`)
- 한국어 설정 (`conversation_language: ko`)
- 개발 모드: hybrid (TDD + DDD)

### 2. MoAI 사용 가이드 문서 작성
- 위치: `docs/moai-guide.md` (743줄)
- 내용: 설치, 명령어, 워크플로우, 에이전트, 품질 프레임워크

### 3. 원본 사이트 분석
- 메인 페이지 및 6개 주요 섹션 분석
- 네비게이션 구조 파악
- 디자인 시스템 (CI 색상) 추출:
  - Yellow (#FFD700) - 긍정의 에너지
  - Orange (#FF8C00) - 따뜻한 온정
  - Green (#228B22) - 마음의 안정
  - Blue (#4169E1) - 내면의 치유

### 4. SPEC 문서 작성
- 위치: `.moai/specs/SPEC-ARTSWISDOM-001.md`
- 내용: 사이트 구조, 디자인 시스템, 컴포넌트 설계, 페이지별 상세

### 5. Next.js 프로젝트 설정
- Next.js 16 + TypeScript + Tailwind CSS 4 초기화
- 추가 패키지 설치:
  - framer-motion (애니메이션)
  - lucide-react (아이콘)
  - embla-carousel-react (캐러셀)
  - react-hook-form + zod (폼 검증)

### 6. 기본 컴포넌트 구현

#### 레이아웃 컴포넌트
- `src/components/layout/Header.tsx` - 반응형 헤더 (드롭다운, 모바일 메뉴)
- `src/components/layout/Footer.tsx` - 푸터 (소셜 링크, 연락처)
- `src/components/layout/Logo.tsx` - 로고 (밭 전(田)자 모티브)

#### 공통 컴포넌트
- `src/components/common/Section.tsx` - 섹션 컨테이너
- `src/components/common/Button.tsx` - 버튼 (4가지 variant)
- `src/components/common/Card.tsx` - 카드 컴포넌트

#### 홈페이지 컴포넌트
- `src/components/home/Hero.tsx` - 히어로 섹션 (애니메이션)
- `src/components/home/MissionVision.tsx` - 미션/비전/가치
- `src/components/home/Programs.tsx` - 프로그램 카드
- `src/components/home/SDGs.tsx` - SDGs 목표 그리드

### 7. 상수 및 설정
- `src/constants/navigation.ts` - 네비게이션 구조, 회사 정보
- `src/constants/colors.ts` - CI 색상 팔레트
- `src/app/globals.css` - 전역 스타일, CSS 변수, 애니메이션

## 프로젝트 구조

```
wisdom/
├── src/
│   ├── app/
│   │   ├── layout.tsx      # 루트 레이아웃
│   │   ├── page.tsx        # 홈페이지
│   │   └── globals.css     # 전역 스타일
│   ├── components/
│   │   ├── common/         # 공통 컴포넌트
│   │   ├── layout/         # 레이아웃 컴포넌트
│   │   ├── home/           # 홈페이지 컴포넌트
│   │   └── ...             # 페이지별 컴포넌트 (예정)
│   └── constants/          # 상수
├── .moai/                  # MoAI 설정
├── docs/                   # 문서
└── public/                 # 정적 파일
```
